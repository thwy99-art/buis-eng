## 02 타깃 사용자

### 핵심 타깃 페르소나

| 구분 | 내용 |
|------|------|
| 연령 / 직군 | 직장 3~7년차, 대기업·중견기업·스타트업 사무직 |
| 영어 수준 | 중급자 — 말은 되지만 표현이 단조롭고 어색함 |
| 핵심 고통 | "말은 하는데 왜 안 통하지?" 뉘앙스·표현 어색함 |
| 사용 환경 | 출퇴근 지하철, 점심시간, 스마트폰 중심 |
| 기존 이탈 이유 | 스픽·말해보카 — 너무 길거나 비즈니스 상황과 무관함 |
| 지불 의향 | 효과 있으면 평생이용권 1만원 고민 없이 결제 |

### 레벨별 사용자 분류

| 레벨 | 특징 | 본 서비스 적합도 |
|------|------|----------------|
| 초보자 | 입이 안 열림, 침묵 회피 | 보조 타깃 |
| 중급자 | 말은 되지만 어색함, 반복 표현 | ★ 핵심 타깃 |
| 고급자 | 유창하나 임팩트 부족 | 2차 타깃 |

## 03 핵심 차별점

| 차별 포인트 | 경쟁사 | 본 서비스 |
|------------|--------|----------|
| 학습 상황 | 일상 회화 / 관심사 기반 | 내 직무 상황만 (디자이너·개발자·마케터·PM) |
| 학습 시간 | 스픽 20~30분 / 엘스 20분 | 하루 3분 완결 |
| 학습 방식 | 발화량 늘리기 | 어색한 표현 → 자연스러운 표현 |
| AI 피드백 | 발음 점수 중심 | 표현 교정 + 한 줄 이유 설명 |
| 가격 구조 | 월 구독 (지속 부담) | 평생이용권 1회 구매 |
| 진입 난이도 | 문장 직접 구성 필요 | 정답/오답 선택지로 시작 |

> **포지셔닝 한 줄 정의**
> "스픽은 말을 많이 시키고, 우리는 말을 제대로 시킨다" — 발화량이 아니라 표현의 질을 올려주는 서비스

## 04 사용자 플로우

### 비회원 플로우
**랜딩 → 직무 선택 → 맛보기 문제 1개 → 완료 화면(가입 유도) → 로그인**

- 가입 없이 바로 체험 가능
- 맛보기 문제는 직무별 전용 풀 3개 안에서만 랜덤 순환
- 완료 후 "다음 패턴" 버튼 비노출, 가입 유도 카드 표시
- 비회원 학습 데이터: 연속일 1일(접속 기준), 학습 문장 최대 3개, 재접속 시 초기화
- localStorage에 학습 기록 저장 안 함 (스트릭·문장수 누적 없음)

### 회원 신규 플로우
**랜딩 → 이메일 입력(매직 링크) → 직무 선택(최초 1회) → 홈 → 학습 시작**

### 회원 재방문 플로우
**접속 → 홈 자동 진입 → 학습 시작 (직무 선택 화면 생략)**

---

## 05 학습 플로우

### Step 1: 알맞은 표현 고르기
- 상황 카드 제시
- 선택지 3개 (정답 1개, 오답 2개) — 매번 랜덤 순서로 셔플
- 선택 전: 한국어 뜻 숨김
- 오답 선택 시: 해당 버튼 빨간 피드백 + 이유 설명, 다시 선택 가능 (정답 미표시)
- 정답 선택 시: 초록 피드백 + 모든 선택지 한국어 뜻 표시

### Step 2: 소리 내어 말하기
- 정답 표현 상단 고정 표시
- 발화 카운트 도트 3개 (3번 말하면 체화 완료)
- 녹음 + 다시 듣기
- Web Speech API로 발음 일치율 피드백 (비슷해요 / 다시 해봐요)

## 06 콘텐츠 구조

### 난이도 3단계

| 단계 | 기준 | 특징 | 목표 정답률 |
|------|------|------|------------|
| 기초 (1~30번) | 문법 오류가 명확한 오답 | 틀린 선택지가 눈에 띄게 어색함 | 70%+ |
| 중급 (31~70번) | 문법은 맞지만 뉘앙스 차이 | 둘 다 말은 되는데 하나가 어색함 | 50~70% |
| 고급 (71~100번) | 셋 다 자연스럽지만 TPO 차이 | 가장 프로페셔널한 표현 선택 | 40~60% |

- 문제 순서 고정 (1→100번, 기초→중급→고급)
- 초반에서 자신감 → 후반으로 갈수록 도전감

### 콘텐츠 목표
직무 4개 × 단계별 100개 = 직무당 100개 (총 400개)

| 직무 | 주요 상황 |
|------|----------|
| 프로덕트 디자이너 | 시안 리뷰, 피드백 요청, 디자인 발표, UX 논의 |
| 개발자 | 코드 리뷰, 스탠드업, 기술 논의, PR |
| 마케터 | 성과 보고, 캠페인 발표, A/B 테스트, ROI |
| PM / PO | 스프린트 플래닝, 로드맵, 우선순위, 에스컬레이션 |

## 07 수익 구조

| 항목 | 무료 | 유료 (평생이용권) |
|------|------|-----------------|
| 광고 | 배너 + 영상 광고 (15회 이상 시 선택형) | 없음 |
| 콘텐츠 | 직무당 100개 전체 | 직무당 100개 전체 |
| 복습 기능 | 없음 | 틀린 문장 자동 복습 |
| 학습 기록 | 저장됨 | 저장됨 |
| 가격 | 무료 | 10,000원 (1회 결제, 영구 이용) |

> **수익화 전략**
> - 콘텐츠로 막지 않고 복습으로 유료 전환 유도
> - 비회원이 맛보기 후 틀린 표현이 쌓이는 게 보여도 복습 불가 → 자연스러운 가입·결제 동기
> - 런칭 특가 프레이밍: "정식 출시 후 월 구독 전환 예정, 지금만 1회 결제"

## 08 기능 명세

### 화면 구조

| 화면 ID | 화면명 | 접근 조건 |
|---------|--------|----------|
| screen-landing | 랜딩 | 비회원 진입점 |
| screen-login | 로그인/가입 | 매직 링크 이메일 방식 |
| screen-job-entry | 직무 선택 | 비회원(맛보기), 회원(최초 1회) 공통 |
| screen-welcome | 홈 | 회원 전용 |
| screen-lesson | 학습 | 비회원/회원 공통 |
| screen-complete | 학습 완료 | 비회원/회원 공통 (내용 분기) |
| screen-payment | 결제 | 회원 전용 |
| screen-mypage | 마이페이지 | 회원 전용 |

### 마이페이지 기능
- 계정 이메일 확인
- 플랜 상태 (무료 / 평생이용권)
- 직무 변경
- 학습 통계 (연속 학습일, 누적 횟수)
- 로그아웃

### 학습 완료 공유 카드
- 정사각형 이미지 (320×320px)
- 포함 내용: 서비스명, 오늘 배운 표현(영어+한국어), 연속 학습일, 오늘 말한 문장, 누적 학습 횟수
- PNG 다운로드 (html2canvas)
- 비회원도 제공 (세션 데이터 기반)

### 광고 구조
- 홈 화면: 카드형 배너 (콘텐츠처럼 자연스럽게)
- 완료 화면: 카드형 배너
- 영상 광고: 15회 학습 이후 선택형 ("광고 보고 복습 잠금 해제")
- 프리미엄 결제 시 모든 광고 즉시 제거
